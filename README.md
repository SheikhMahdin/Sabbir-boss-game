🎮 Sabbir Boss Game
A fast-paced, interactive web-based clicker game where you defend Sabbir (the best teacher of MISC!) from incoming mosquitoes before they swarm him!
🕹️ About the Game
Click mosquitoes to eliminate them before they reach Sabbir's face. Watch his expression change from happy to angry as mosquitoes get closer, and try to defeat as many waves as possible. Features progressive difficulty, combo scoring, and satisfying splat effects!
✨ Features

🦟 Animated Mosquitoes - Detailed, realistic mosquito animations with flapping wings and hovering behavior
😊 Dynamic Expressions - Sabbir's face changes from happy to angry based on mosquito proximity
🔥 Combo System - Build combos for bonus points when killing mosquitoes consecutively
📈 Progressive Waves - Difficulty increases over time with more mosquitoes spawning
🎵 Music Control - Toggle background music on/off
💥 Splat Effects - Satisfying visual feedback when eliminating mosquitoes
📱 Fully Responsive - Works seamlessly on desktop, tablet, and mobile devices
🌈 Beautiful UI - Gradient backgrounds, glowing elements, and smooth animations

🛠️ Technologies Used

HTML5
CSS3 (Animations, Gradients, Flexbox)
Vanilla JavaScript (No dependencies!)

🚀 Getting Started

Clone the repository
Replace image sources (happy-sabbir.png and angry-sabbir.png) with your own images
Optionally add a background music file (background-music.mp3)
Open index.html in a web browser

📊 Game Mechanics

Base Score: 10 points per mosquito
Combo Bonus: 5 points × combo multiplier
Game Over: When 10 mosquitoes reach Sabbir
Wave System: Difficulty increases every 10 spawns
